This index.html exists mainly to test WhatsApp previews. When this URL is shared on WhatsApp, the preview image and text come from the Open Graph meta tags in the <head>.

How to use

  1. Host this file on GitHub Pages
  2. Replace <username> and <repo-name> in og:url
  3. Share the page URL on WhatsApp

Task pages

For per-task previews, create pages like:

  • /task/123.html
  • /task/124.html

Each task page should have its own og:image pointing to the uploaded screenshot.
